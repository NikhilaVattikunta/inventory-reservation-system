# Smart Inventory Reservation System

A full-stack inventory reservation system built using Next.js, Prisma ORM, Neon PostgreSQL, and Vercel deployment.

## Features

- Real-time inventory tracking
- Product reservation system
- Reservation confirmation API
- Automatic stock management
- REST APIs with Next.js
- PostgreSQL database using Prisma ORM
- Live deployment on Vercel

## Tech Stack

- Next.js 16
- TypeScript
- Prisma ORM
- Neon PostgreSQL
- Vercel
- Thunder Client

## API Endpoints

### Get Products
GET /api/products

### Create Reservation
POST /api/reservations

### Confirm Reservation
POST /api/confirm
